Notebook: Detección de muérdago con SVM y Descriptores de textura

Este notebook implementa un flujo de trabajo de aprendizaje supervisado para detectar muérdago en imágenes multiespectrales capturadas por dron, utilizando descriptores de textura basados en GLCM (Gray Level Co-Occurrence Matrix).

Objetivo

Evaluar la capacidad discriminativa de diferentes combinaciones de bandas y texturas para la clasificación pixel a pixel de áreas infestadas por muérdago, mediante un clasificador SVM (Support Vector Machine).

Descripción del flujo

  • Preprocesamiento: Normalización de las bandas seleccionadas mediante MinMaxScaler (rango 0 a 1).
  • Vectorización: Conversión de las bandas y máscaras a formato vectorial para entrenamiento y evaluación.
  • Entrenamiento: Entrenamiento del clasificador SVM con los datos de textura generados a partir de imágenes etiquetadas. Definición de hiperparámetros con GridSearch.
  • Evaluación: Reporte de precisión, Kappa, F1-score, y visualización de predicciones para validación visual.

Características clave

  • Uso de librerías como scikit-learn, skimage, NumPy, Pandas.
  • Implementación de GLCM para extraer patrones de textura de las imágenes multiespectrales.
  • Evaluación rigurosa de métricas de desempeño (accuracy, Kappa, precision, recall).

Entorno de ejecución

  • Plataforma: Google Colab
  • Lenguaje: Python
  • Dependencias clave: scikit-learn, skimage, matplotlib, pandas
